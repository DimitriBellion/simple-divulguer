# Divulger Bot

A simple bot divulger, use for send all members in your servers. Use at moderation.

## Clone Repository
```
$ git clone https://github.com/DimitriBellion/simple-divulguer.git
``` 

## Using

```
$ cd simple-divulguer
$ yarn or npm install
```

## Change variables in arquives 
- index.js
- commands/div.js

## Starting Bot
```
$ yarn dev
```
